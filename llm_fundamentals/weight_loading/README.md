# ⚖️ Weight Loading

Load and use pre-trained GPT weights.

## Files

| File | Description |
|------|-------------|
| `01_weight_loading.py` | Load weights |
| `02_run_inference.py` | Run inference |
| `03_gpt_download.py` | Download weights |
| `04_weight_helpers.py` | Helper functions |

## What You'll Learn

- Download pre-trained weights
- Load weights into model
- Run inference
- Generate text
- Understand model outputs

## Run It

```bash
python 01_weight_loading.py
python 02_run_inference.py
```

## Download Weights

The `03_gpt_download.py` script downloads GPT-2 weights automatically.
