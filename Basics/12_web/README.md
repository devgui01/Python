Build web applications with Flask.

## Files

| File | Description |
|------|-------------|
| `flask_hello_world.py` | Hello World with Flask |

## Quick Start

### Install Flask
```bash
pip install flask
```

### Run the Server
```bash
python flask_hello_world.py
```

### Visit
```
http://127.0.0.1:5000
```
